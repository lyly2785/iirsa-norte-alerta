🚀 Nuevas funcionalidades — Versión 2.0 (integradas al flujo)
🔔
Push Notifications Geolocalizadas
Se insertan en Etapa 1b. Firebase Cloud Messaging + Geofencing de 20 km. Solo eventos MODERADO/GRAVE. El usuario no necesita tener la app abierta.
🎤
Editor de Voz STT (Speech-to-Text)
Se activa en Etapa 1, campo Descripción. Google Speech API o Whisper offline. El usuario dicta — el sistema transcribe. Reduce 60% el tiempo de reporte. Disponible en zonas sin cobertura con modelo local.
🔗
Sync Gemelo Digital BIM/GIS
Se inserta en Etapa 4. Cada evento queda georreferenciado sobre el modelo 3D del corredor IIRSA Norte y alimenta el GMAO Central de OSITRAN con trazabilidad completa.
⛽
Directorio de Servicios en Ruta
Nueva pestaña independiente al flujo de incidencias. Grifos (precio combustible API OSINERGMIN), restaurantes, hoteles, talleres mecánicos con disponibilidad en tiempo real.
